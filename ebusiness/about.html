Salesforce

SaaS is where users/customers can gain access to applications online- as a service.

one of the main benefits of this is being able to access your projects whenever you like and where ever you are, all you need is a computer and internet.

the payment is normally per month, and calculated on usage. sometimes a fixed fee is available.

Cloud 9

Cloud 9 provides a computing platform and/or solution stack typically including operating system, programming language execution environment, database, and web server.

application developers can develop and run their software solutions on a cloud platform without the cost and complexity of buying and managing the underlying hardware and software layers.

some PaaS offers, the underlying compute and storage resources scale automatically to match application demand such that the cloud user does not have to allocate resources manually.
